# Import Staging

Use this folder for files you import manually before the frontend implementation is wired up.

## Folders

- `source-zip/`: put the frontend prototype zip here.
- `figma-images/`: put screenshots, PNGs, JPGs, and visual references here.
- `figma-exports/`: put exported SVGs, design tokens, or structured Figma output here.

After importing, move production source files into `frontend/src/` and public static assets into `frontend/public/`.
